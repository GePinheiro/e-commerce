🍧 Site de Açaiteria

Este é um projeto de site para uma açaiteria, desenvolvido para apresentar o cardápio, divulgar produtos e facilitar o contato com os clientes. O objetivo é oferecer uma experiência simples e rápida para quem deseja conhecer e pedir açaí e acompanhamentos.

📌 Funcionalidades

🥣 Apresentação da açaiteria

📋 Exibição do cardápio de açaí

🍓 Lista de acompanhamentos (granola, leite condensado, frutas, etc.)

📱 Design responsivo para celular e computador

📞 Informações de contato e redes sociais

📍 Localização da loja

🛠️ Tecnologias Utilizadas

HTML5 – Estrutura do site

CSS3 – Estilização e layout

JavaScript – Interatividade

Bootstrap (opcional) – Responsividade
